# fake连接器
上级文档: [connectors](../introduction_zh.md)

***BitSail*** fake连接器支持批式读，其支持的主要功能点如下
- 随机生成测试数据，支持unique,指定范围等特性

## 依赖引入
```xml
<dependency>
   <groupId>com.bytedance.bitsail</groupId>
   <artifactId>bitsail-connector-fake</artifactId>
   <version>${revision}</version>
</dependency>

```
